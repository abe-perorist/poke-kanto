# 朝遅め出発プラン（13:00フォレスト枠を活かしたゆっくり版）

フォレスト入場が 13:00 固定のため、9:00 開園に合わせて急ぐ必要がないことを活かしたプラン。  
**だいすきショップの買い物は夕方に回す**ことで、午前中の動きをシンプルにする。

## 朝遅め版タイムライン

> **このプランが機能する理由：**  
> - フォレストは 13:00 固定なので開園前に並ぶ必要がない  
> - だいすきショップの整理券は夕方枠をエントリーして、夕方にまとめて買い物  

### 〜 午前：ポケパーク入場 → タウン探索 〜

| 時刻 | 行動 | ポイント |
|------|------|---------|
| **08:37**（または 08:21） | **大倉山駅 出発** | ルート1 推奨。ルート2 なら16分早着 |
| **09:26**（または 09:15）〜 | よみうりランド到着 → 入園 | 10:00 開園前に着くのでゲート前で少し待つ形になる |
| **10:30頃** | ポケパーク カントー入場 → **整理券を即エントリー** | だいすきショップ（先着・**夕方の遅い枠**）＋ピカピカスパーク（抽選） |
| 10:30〜11:00 | ポケモントレーナーズマーケット → ピンズ購入 | 朝一より落ち着いた環境で買いやすい |
| 11:00〜12:30 | ランチ・ポケモンセンター・ストリートグリーティング・ワゴン類 | ディグダワゴン・ニャースワゴン等。11時前後がランチのベストタイミング |
| （抽選当選時のみ）**11:25〜** | **ピカピカスパーク 入場待ち** | 11:45〜25分間 |
| 12:30〜13:00 | 軽食 + **フォレスト前トイレ**（子ども必須） | **フォレスト内にトイレなし** |

## 整理券エントリー方針（朝遅め版）

| 担当 | エントリー内容 | 備考 |
|------|--------------|------|
| どちらか1人 | **ポケモンだいすきショップ**（先着・**夕方の遅い枠**） | 17:00以降の枠を意識してエントリー |
| 落ち着いたら | ピカピカスパーク（抽選） | 11:45の回が唯一フォレスト前に消化できる |

> だいすきショップの整理券さえ押さえれば、あとはゆっくり動ける。

## 早出しプランとの比較

| 比較点 | 早出し（07:21発） | 朝遅め（08:37発） |
|-------|-----------------|-----------------|
| 大倉山出発 | **07:21** | **08:37**（ルート2なら08:21） |
| よみうりランド着 | **08:20〜**（開園前から並ぶ） | **09:26〜**（9:00開園前に着いて少し待つ） |
| 整理券エントリー | ふれあいハウス＋だいすきショップ等を争奪 | **だいすきショップのみ**・シンプル |
| だいすきショップ | 午前 or 夕方どちらでも | **夕方に集中**（落ち着いて買える） |
| フォレスト滞在 | 60〜90分 | **2時間・ゆっくり楽しむ** |
| パレード（15:50） | 余裕あり | フォレスト退出後ちょうど間に合う |
| 子どもの体力 | 9時間以上いるので終盤疲弊 | 約7〜8時間・余裕あり |
