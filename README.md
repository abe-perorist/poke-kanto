# ポケパーク カントー 家族旅行プラン 🎮

4人家族（大人2名・小3男子・小1女子）でポケパーク カントーを満喫するためのプランと準備タスクを管理するリポジトリです。

---

## 基本情報

| 項目 | 内容 |
|------|------|
| チケット | トレーナーズパス（フォレスト+タウン両エリア対応） |
| フォレスト入場時刻 | **13:00〜** |
| 会場 | よみうりランド遊園地内（東京都稲城市） |

---

## アクセス（大倉山駅から）

```
大倉山駅（東急東横線）
  ↓ 約15分
渋谷駅（JR山手線/中央線に乗換）
  ↓ 約5分
新宿駅（京王線に乗換 ※京王新宿駅）
  ↓ 約15分（特急）
京王よみうりランド駅
  ↓ ゴンドラ「スカイシャトル」約10分（片道300円・往復500円）
     またはバス約5分
ポケパーク カントー 到着
```

**合計所要時間：約50〜60分**

## 整理券エントリー 優先順位

入園直後にアプリで以下の順に確保する。午前中に埋まるため**入園後10分が勝負**。

1. 🏠 **ポケモンふれあいハウス**（最優先・最激戦）
2. 🎠 **ブイブイヴォヤージュ** または **ピカピカパラダイス**
3. 🏆 **カヤツリジム バトルショー**
4. 🛍️ **ポケモンだいすきショップ**

## 子連れ チェックポイント

- [x] お子さんは2人とも5歳以上 → フォレスト入場OK
