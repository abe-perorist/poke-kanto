## 子連れ おすすめ体験

- **ポケモンセンター**：お気に入りのポケモンぬいぐるみを持参すると、ジョイさんに回復してもらえる特別体験ができる
- **ストリートグリーティング**：整理券がなくてもカヤツリタウン内に常時ポケモンが散歩していて触れ合い可能
- **研究ノート**（フォレスト）：ポケモンの生態を観察するノート。曜日や天候で出現ポケモンが変わる
- **名入れキーホルダー**：デカヌチャンのワゴンで作製可能
- **バースデーシール**：誕生日の前後半年以内ならスタッフにもらえる
