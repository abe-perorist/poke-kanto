## 整理券エントリー 分担作戦

**よみうりランド入園直後（9:00〜）に大人2人で即分担。10分が勝負。**

| 担当 | エントリーする施設 | 種別 |
|------|-------------------|------|
| 大人A | ポケモンふれあいハウス（最優先・最激戦） | 先着 |
| 大人B | ポケモンだいすきショップ | 先着 |
| 落ち着いたら | カヤツリジム ピカピカスパークス | 抽選 |
| 続けて | ブイブイヴォヤージュ / ピカピカパラダイス | — |

> アプリはログイン済み状態にしておく。子どもは安全な場所で待機。

### 事前に必須：アプリでグループ作成＆チケット共有

家族のチケットが別々に購入されている場合、アプリで**グループ作成＋チケット共有**を事前にやっておかないと、代表者が家族全員分の整理券を一括取得できない。当日の混乱を防ぐため**訪問前日までに完了**させる。

## 子連れ おすすめ体験

- **ポケモンセンター**：お気に入りのポケモンぬいぐるみを持参すると、ジョイさんに回復してもらえる特別体験ができる
- **ストリートグリーティング**：整理券がなくてもカヤツリタウン内に常時ポケモンが散歩していて触れ合い可能
- **研究ノート**（フォレスト）：ポケモンの生態を観察するノート。曜日や天候で出現ポケモンが変わる
- **名入れキーホルダー**：デカヌチャンのワゴンで作製可能
- **バースデーシール**：誕生日の前後半年以内ならスタッフにもらえる

## 注意事項まとめ

| 項目 | 内容 |
|------|------|
| キャッシュレス | 完全キャッシュレス。ただしコインロッカーは現金が必要な場合あり |
| 再入場 | カヤツリタウンはスタンプで再入場OK。フォレストとだいすきショップは再入場不可 |
| フォレスト トイレ | **エリア内にトイレなし**。入場前に必ず済ませる（特に子ども） |
| フォレスト 装備 | 5歳未満入場不可。階段110段・急斜面あり。スニーカー必須。サンダル・ヒール厳禁 |
| ゴンドラ | 朝早い時間は未運行の場合あり。バスで代替可 |
| モバイルバッテリー | 終日アプリ使用＋QR決済＋写真で消耗激しい。**1万mAh以上持参推奨**（レンタルは午後に在庫切れ多い） |
| だいすきショップ | レジ待ち1〜1.5時間かかる場合あり。会計1回限りのため事前に欲しいものを全部決める |
| 食事の席 | 座席が少なく昼時は確保困難。時間をずらすか立食を覚悟 |
| ロッカー | ポケパーク内になし。よみうりランド側まで往復約20分かかる。荷物は1回でまとめて預ける |
| 無料Wi-Fi | 園内フリーWi-Fi あり。アプリ利用のため接続設定しておく |
| バースデーシール | 誕生日前後半年（実質年中）もらえる。スタッフに申し出る |
